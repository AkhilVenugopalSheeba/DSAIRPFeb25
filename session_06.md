# Python Data Types
## Numeric Type
 The values that are entered in numbers in python.
 
### Integer
 Integer are whole numbers which are whole numbers,positive or negative numbers.

### Float
Float are numbers that have decimal points.

### Complex
 Complex are numbers that have real and imaginary part.
 
## Text Type
### String
 Characters which are enclosed in (' or ").
 
## Sequence Type
### List

 Lists are characters that are ordered,changeable and allows duplicates.
 
### Tuple
 Tuples are characters that are not changeable but are ordered and allows duplicates.
 
### Range
 Range is which used to generate a sequence of numbers.

## Mapping Type
### Dictionary
 Dictionay is used to store values in pairs.

## Set Type
 Set is an unordered,mutable values that do not allow duplicates.

## Boolean
 Represents a True or a False statements.

## None Type
 It is a type which represents the absense of a value.
